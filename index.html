FAQs

How does URL shortening work? +
URL shortening works by taking a long URL and creating a shorter, condensed version that redirects to the original URL. When a user clicks on the shortened link, they are redirected to the intended destination.
Is it necessary to create an account to use the URL shortening service? +
While you can create shortened URLs without an account, creating one allows you to track link performance, customize URLs, and manage all your shortened links in one place.
Are the shortened links permanent? Will they expire? +
Free plan links remain active for 30 days. Premium plan links are permanent and will not expire unless manually deleted by the user or flagged for violation of terms.
Are there any limitations on the number of URLs I can shorten? +
Free users can shorten up to 50 URLs per month. Professional and Team plans offer unlimited URL shortening capabilities along with additional features.
Can I customize the shortened URLs to reflect my brand or content? +
Yes, premium users can create custom branded links using their own domain name and customize the URL slug to match their brand or content description.
How are you protecting against spam or malicious activity? +
We employ advanced security measures including link scanning, spam detection, and automated monitoring systems. All links are checked against known malware databases, and suspicious activities are flagged for review.
What is a QR code and what can it do? +
A QR code is a two-dimensional barcode that can be scanned by smartphone cameras to quickly access websites or information. Our service generates QR codes for your shortened URLs, making it easy to share links in print materials or physical locations.
Is there an API available for integrating the URL shortening service into my own applications or websites? +
Yes, we offer a comprehensive API for developers to integrate our URL shortening service. The API is available to Professional and Team plan subscribers, with documentation and support resources available.
